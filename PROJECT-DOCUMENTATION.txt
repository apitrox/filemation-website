================================================================================
PROJECT OVERVIEW
================================================================================

This project is a complete recreation of the Filemation.com website 
(electronic document management system) as a modern, static HTML website 
using Tailwind CSS.

The original website files were located in the filemation-website-files 
directory and have been recreated from scratch with modern web technologies 
while maintaining the exact design, layout, and color scheme.

================================================================================
JAVASCRIPT FUNCTIONALITY
================================================================================

The js/main.js file implements the following features using vanilla JavaScript:

1. HEADER BEHAVIOR
   - Fixed positioning on scroll
   - Adds "header-fixed" class when scrolled > 100px
   - Header shrinks from 92px to 72px height
   - Logo scales appropriately

2. BACK TO TOP BUTTON
   - Appears when scrolled > 100px
   - Smooth scroll animation to top
   - Fade in/out transitions

3. MOBILE NAVIGATION
   - Clones desktop nav menu for mobile
   - Creates hamburger toggle button
   - Creates overlay for background
   - Toggle between hamburger (fa-bars) and close (fa-times) icons
   - Slide-in animation from left
   - Close on outside click
   - Close on menu item click

4. SMOOTH SCROLLING
   - Smooth scroll for anchor links
   - Accounts for fixed header height
   - Updates active menu item

5. PRICING PAGE TOGGLE
   - Annual/Monthly billing switch
   - Toggles display of .m and .y elements
   - Changes prices dynamically

6. FORM HANDLERS (Non-functional)
   - Login form: Simulates validation, shows error
   - Register form: Simulates processing, shows error
   - Contact form: Simulates sending, shows success message

7. SCROLL ANIMATIONS
   - Intersection Observer API for fade-in effects
   - Observes elements with [data-animate] attribute
   - Triggers animations when elements enter viewport

All jQuery dependencies have been removed and replaced with modern vanilla
JavaScript using features like:
- addEventListener
- querySelector/querySelectorAll
- classList API
- Intersection Observer
- Smooth scrolling with window.scrollTo
- setTimeout for simulated delays

CSS ARCHITECTURE
================================================================================

The css/style.css file uses Tailwind CSS with custom components:

STRUCTURE:
1. Google Fonts import
2. Tailwind directives (@tailwind base, components, utilities)
3. @layer base - Global element styles
4. @layer components - Reusable component classes

KEY COMPONENTS:
- .back-to-top - Fixed back to top button
- #header - Fixed header with transitions
- #hero - Hero section with background image
- .nav-menu - Desktop navigation styles
- #mobile-nav - Mobile navigation drawer
- #features - Feature boxes with icons
- #footer - Footer styles
- .btn - Button styles
- .form - Form input styles

RESPONSIVE DESIGN:
- Mobile-first approach
- Breakpoint at 768px for tablet/desktop
- Hamburger menu below 768px
- Grid layouts adapt to screen size

MAINTENANCE NOTES
================================================================================

To Update Content:
1. HTML files can be edited directly
2. Update text, links, images as needed
3. Tailwind classes can be added/modified inline

To Change Colors:
1. Update tailwind.config.js
2. Update inline config in each HTML file
3. Colors are defined in one place for consistency

To Add New Pages:
1. Copy an existing HTML file as template
2. Update navigation in all pages to include new link
3. Follow the same header/footer structure

To Modify Forms:
1. Form handlers are in js/main.js
2. Update logic in event listeners
3. Can connect to actual backend API if needed

To Update Images:
1. Replace files in /images directory
2. Keep same filenames or update HTML references
3. Maintain aspect ratios for best results
